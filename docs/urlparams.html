Url parameters

When linking to Bassoontracker you can use the following urls parameters

file   Url to a mod you want to open initially when Bassoontracker gets loaded.
Ideally it should be a HTTPS links.
Bassoontracker will proxy HTTP links through it's own server but this will inevitably slower.

Example

autoplay   This will (try to) autoplay the loaded file.
Please note that most browsers will block this and only will play audio on user interaction.
You can change this behaviour through your browser settings to allow audio for a specific domain.
