F.A.Q.

Dude, stereo is not working

You are using Safari, right?
The technical story:
Safari doesn't support StereoPanner so everything will be in mono.
It does support a normal Panner, but you can't set their position "over time" so they are somewhat useless.
Yes, I know there's a nice shim. This works but it is too slow.
The solution:
Use another browser like Chrome or Firefox. There you'll have full stereo.
