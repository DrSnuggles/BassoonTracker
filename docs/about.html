About

Bassoon Tracker is a free music creation app following the long tradition of so called "tracker" music programs.

It runs in your browser and produces realtime generated music, using samples, notes and effects.
Bassoon Tracker handles the popular .mod and .xm file formats.

Most notable features are:
  • 100% plain old javaScript - no dependencies
  • Open Source
  • Protracker and Fasttracker compatible: load, edit and save .mod and .xm files
  • up to 32 channels
  • build-in sample editor
  • Import 8bit WAV, 8SVX and RAW samples (as well as any other format AudioContext can read, like .mp3 and .ogg
  • Includes the historic ST-01 and ST-02 sample disks, released in 1987 (as well as a bunch of others)
  • The modarchive API is integrated to access many thousands of music files
  • The modules.pl API is integrated to access even more music files
  • Export to .mod, .xm, .wav and .mp3
  • Connect your Dropbox account to directly read and write your own files
  • Includes a lightweigh player library you can use in your own projects
